Y.C

Toyota Hawaii Care Claim automated validation




Overview

Due to the COVID, Servco was low in staff and when doing validation for claim for the Toyota Hawaii Care is slow and have mistake. So, Servco decided to create an automate system that will validate the claim and process it. The system is using api call to run sql sproc that doing all the validation and processing it if it passes all validation.



What I contributed to the project

In the project, I was the one of the two software engineer. I mainly focus on creating all the sql sproc that doing the validation base on what the claim manager told us. For example, the car under the claim number have to be bought in Hawaii less than 2 year. Than I will create a sql sproc that will take in the claim number and check database if the purchase location is hawaii and compare the current time to see if the car is bought less than 2 year.
